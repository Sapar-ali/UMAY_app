## 🚀 Запуск:

### 1. Активируйте виртуальное окружение:
```bash
source .venv/bin/activate
```

### 2. Запустите приложение:
```bash
python app.py
```

Или используйте удобный скрипт:
```bash
python run.py
```

### 3. Откройте в браузере:
**http://localhost:5001**
